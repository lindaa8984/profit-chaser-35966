# إعداد حساب Try التجريبي

## نظرة عامة
حساب Try هو حساب تجريبي يسمح للمستخدمين بتجربة النظام دون التأثير على البيانات الفعلية. جميع البيانات التي يتم إنشاؤها باستخدام هذا الحساب يتم حذفها تلقائياً عند تسجيل الخروج.

## خطوات إنشاء حساب Try

### الطريقة 1: عبر واجهة التسجيل
1. افتح صفحة التسجيل في التطبيق
2. أدخل البيانات التالية:
   - **البريد الإلكتروني**: `try@demo.local`
   - **كلمة المرور**: `Admin@123`
   - **الاسم الكامل**: `Try`
3. أكمل عملية التسجيل

### الطريقة 2: عبر Lovable Cloud Dashboard
1. افتح Lovable Cloud Dashboard
2. انتقل إلى قسم Authentication
3. اضغط على "Add User"
4. أدخل البيانات:
   - **Email**: `try@demo.local`
   - **Password**: `Admin@123`
   - **Confirm Email**: قم بتفعيل هذا الخيار
5. احفظ المستخدم الجديد

### الطريقة 3: عبر SQL في Database
```sql
-- إضافة المستخدم في auth.users
-- ملاحظة: قد لا تعمل هذه الطريقة بسبب قيود Supabase
-- يُفضل استخدام الطرق السابقة

INSERT INTO auth.users (
  email,
  encrypted_password,
  email_confirmed_at,
  raw_user_meta_data
) VALUES (
  'try@demo.local',
  crypt('Admin@123', gen_salt('bf')),
  now(),
  '{"full_name": "Try"}'::jsonb
);
```

## المميزات
- **بيئة منعزلة**: لا يرى بيانات أي مستخدم آخر
- **حذف تلقائي**: جميع البيانات تُحذف عند تسجيل الخروج
- **تجربة كاملة**: إمكانية الوصول لجميع المميزات

## العنوان المعروض
عند تسجيل الدخول بحساب Try، سيظهر العنوان:
**DIG لإدارة العقارات — إدارة متكاملة**

## ملاحظات مهمة
1. البيانات مؤقتة ولن يتم حفظها
2. الحساب آمن ومعزول عن المستخدمين الآخرين
3. يمكن لأي شخص استخدام هذا الحساب للتجربة
4. البيانات تُحذف عند تسجيل الخروج (signOut)

## كيفية الاستخدام
1. سجل الدخول باستخدام:
   - Email: `try@demo.local`
   - Password: `Admin@123`
2. أنشئ عقارات، عملاء، عقود، ومدفوعات تجريبية
3. جرب جميع المميزات
4. عند الانتهاء، سجل الخروج لحذف جميع البيانات التجريبية
